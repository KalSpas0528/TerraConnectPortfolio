Project Title & Description:
TerraConnect: Global Interactive News Map
TerraConnect is designed as an interactive platform where users explore a world map to receive localized news updates. Combining dynamic data fetching with a user-friendly interface, this project aims to provide a comprehensive view of global events.

Research & Planning Documentation:
Detailed research was conducted on various mapping libraries (such as Leaflet.js) and news integration methods (including News API, web scraping, and RSS feeds). Wireframes and planning documents were created to outline user experience and technical requirements.

Current Progress with Screenshots & Code Snippets:

Below is an example snippet of the live news fetching function: 

// Example function to fetch live news data
async function fetchNews(country) {
  try {
    const response = await fetch(`api/news?country=${country}`);
    const data = await response.json();
    return data;
  } catch (error) {
    console.error("Error fetching news:", error);
  }
}
