Current Project: TerraConnect

Description: TerraConnect is an interactive world map that provides real-time global news updates. It leverages mapping technology alongside dynamic data fetching to create an immersive and informative user experience.

Research & Planning Documentation

This section outlines the research and planning behind TerraConnect. Key focus areas include:

  • Evaluation of mapping libraries (e.g., Leaflet.js) for optimal interactive performance.
  • Comparison of news integration methods: News API, web scraping, and RSS feeds.
  • User experience research, including wireframes and UI planning.

  } catch (error) {

  }
