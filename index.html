TerraConnect: Global Interactive News Map

TerraConnect is an interactive platform that combines dynamic live features with a global map. Users can explore countries, access live news, view country comparisons, and enjoy engaging educational tools. Powered by Leaflet.js, GNews API, and a user-friendly interface, TerraConnect offers an innovative way to learn about the world.

Key Features

  • Interactive World Map: Click on any country to access live Wikipedia summaries and news updates.
  • Country Comparison: Compare countries side by side in terms of population, area, and more.
  • Live News Feeds: Integrated with the GNews API for real-time news updates.
  • Country Guessing Game: A fun and educational game where users guess countries based on the map.
  • Junior Explorer Mode: A child-friendly mode that focuses on educational content with restricted news access.
  • Dark/Light Mode Toggle: Users can switch between dark and light modes for better accessibility.

Project Timeline

Week 1 - Initial Setup

TerraConnect started with the implementation of an interactive map using Leaflet.js, enabling users to click on any country and view its Wikipedia summary. The foundational work focused on mapping and setting up the API structure for future features.

Week 6 - Country Guessing Game & Local Storage

The Country Guessing Game feature was added, allowing users to challenge themselves by guessing countries based on the map. I also integrated local storage to track user progress, such as countries visited in the game.

Week 16 - Junior Explorer Mode & Final Touches

The Junior Explorer Mode was introduced, providing an educational experience for children with a focus on learning through fun facts and interactive exploration. The project was polished with bug fixes, and the news feature was fully optimized for better performance.

Challenges & Solutions

Throughout the development process, several challenges were overcome:

  • News Integration: Initially, I tested different methods for integrating live news, including APIs, web scraping, and RSS feeds. Ultimately, I chose the GNews API for its reliability and ease of integration.
  • Performance Optimization: Ensuring that the map was responsive and that the app performed well on both desktop and mobile required balancing rich interactivity with smooth user experience.
  • Junior Explorer Mode: Developing a safe and engaging experience for younger users required careful planning and restructuring of the application architecture.
