TerraConnect: Global Interactive News Map

TerraConnect is a fully client-side web platform that empowers users to explore the world through an interactive map. Features range from educational tools to live content feeds—designed for everyone from geography enthusiasts to curious kids.

📌 Feature Walkthrough

🌍 Interactive World Map

Click any country on the Leaflet-powered map to view live Wikipedia summaries. These popups act as gateways to deeper interaction: learning sessions, comparisons, and real-time news.

📰 GNews-Powered Live News Feeds

By integrating the GNews API, users can access the latest headlines for any selected country. This turns TerraConnect into not just an explorer—but a live news companion for international events.

🎮 Country Guessing Game

A game mode challenges users to guess countries on the map. Visual feedback and answer validation help users sharpen their geographic memory. Your session history is saved using localStorage.

👦 Junior Explorer Mode

Perfect for kids, this simplified version of the map offers educational facts, hides complex features like news feeds, and includes engaging visuals. The goal is to spark curiosity and foster geographic learning early.

📘 Learn Feature

Every country has a dedicated learn modal that includes a mini-map, language breakdown, population, and other fascinating facts. Perfect for students and educators alike.

🌓 Dark Mode

Toggle between light and dark themes to reduce eye strain. The application uses CSS variables to apply a full theme swap without affecting usability or performance.

✅ Visited Countries Tracker

Clicking on countries adds them to your visited list. The list is persistent via localStorage, and can be cleared with a single click—giving users a sense of progress and revisit history.

Challenges & Solutions

  • News Integration: Tried various APIs and even RSS, but settled on GNews for stability and coverage.
  • Mobile Performance: Leaflet map tuning and UI throttling ensured smooth rendering on all devices.
  • Kid-Friendly Architecture: Junior Explorer mode involved conditional rendering, interface rework, and local logic toggles.
